Project Overview: Fighting Game Development


This project focuses on the design and development of a Tekken-inspired third-person fighting game using the Unity game engine. The objective of the project is to implement and understand the core mechanics of a 3D fighting game, including player movement, combat interactions, health management, animations, sound effects, and scene control. The game targets the PC platform and emphasizes smooth gameplay and responsive controls rather than monetization.

The game includes multiple playable scenes/maps, each offering a distinct environment while maintaining consistent gameplay mechanics. Both player and opponent characters display real-time health bars that update dynamically based on damage received during combat. A structured combat system manages attack execution, hit detection, damage calculation, and reaction animations, ensuring a balanced and engaging fighting experience.



The project integrates sound effects and visual feedback to enhance immersion. Hit sounds, attack effects, and animations are synchronized to provide immediate feedback to the player while avoiding excessive or overlapping audio triggers. Unity assets are effectively utilized and customized to speed up development while maintaining visual and performance quality.

Overall, this project strengthens practical knowledge of game design principles, Unity’s component-based architecture, C# scripting, animation systems, AI behavior design, UI integration, and debugging techniques. It demonstrates the ability to design and implement a complete interactive combat system and reflects a growing proficiency in real-time game development.
